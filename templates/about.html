About Style Transfer

Introduction

Image style transfer is a subject that has been studied a lot in the past few years. However, the existing models focus on transferring the whole image to a designated style. While learning more about machine learning in AMLI, computer vision is one area that we havenâ€™t had enough chance to explore. Style transfer is one subject that has the potential to be impactful and useful to a large audience. With current deployment of some apps, one can apply artistic filters to the photos. We want to create a web app that interactively offers more style transfer options to users.

The style transfer web app is designed to process the image style transfer based on two input images- one is the style reference image (extracting the style from this image) and the second one is the image that embeds the style onto itself (creating an artistic effect on the chosen image).

Objective

The objective of this team is to build a successful web app that not only processes a style transfer onto an image, but also allows the user to select specific objects in the image to process the transfer and other artistic special effects.

Process

The three main processes are: object detection/classification/segmentation, basic style transfer, and performing style transfer on specific objects. The model trains on the COCO dataset. Since this project idea has been very well studied, using pretrained models is a great approach. The obeject detection uses the RCNN model, and the style transfer uses keras and VGG19 model for image classification.
